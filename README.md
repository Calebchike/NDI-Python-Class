# NDI Python Essential Training

This is the repository for the **NDI Python Essential Training** course.

Python is one of the most commonly used dynamic languages for large organisations including Google, Yahoo, and IBM. In this course, senior software engineer **Okwunna Olive** guides you through all the essentials of learning and using Python, from installation and basic syntax through to classes, error handling, concurrency, and file I/O.
